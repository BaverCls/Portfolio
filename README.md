# 🚀 Modern Portfolio Project

Bu proje, modern bir kullanıcı arayüzü ile güçlendirilmiş, yönetilebilir bir portfolyo web uygulamasıdır. Hem tasarım hem de performans odaklı geliştirilmiş olan bu çalışma, projelerin dinamik olarak sergilenmesini ve bir yönetim paneli üzerinden kontrol edilmesini sağlar.

---

## 🎨 Tasarım Özellikleri
- **Modern UI/UX:** Karanlık mod odaklı, şık ve minimalist bir tasarım.
- **Dinamik Slider:** Projeler arasında akıcı geçişler sağlayan, hover efektli ve mobil uyumlu slider.
- **Etkileşimli Bileşenler:** Framer Motion ile güçlendirilmiş mikro animasyonlar.

## 🛠️ Teknik Stack
### Frontend
- **React + Vite:** Hızlı ve modern bir geliştirme ortamı.
- **Tailwind CSS:** Esnek ve responsive tasarım dili.
- **Framer Motion:** Gelişmiş animasyonlar.

### Backend
- **FastAPI:** Yüksek performanslı Python tabanlı API.
- **SQLAlchemy:** Veritabanı yönetimi (ORM).
- **Supabase PostgreSQL:** Güvenilir ve hızlı veritabanı.
- **Supabase Storage:** Kalıcı ve CDN destekli görsel depolama sistemi.
- **JWT Authentication:** Admin paneli için güvenli giriş sistemi.

---

## 🚀 Öne Çıkan Özellikler
- **Admin Paneli:** Projeleri kolayca ekleyebilir, güncelleyebilir veya silebilirsiniz.
- **İletişim Formu:** Ziyaretçilerin size ulaşmasını sağlayan entegre iletişim formu.

---

## 🌐 Yayına Alma (Deployment)
- **Backend:** DigitalOcean App Platform / Droplet.
- **Frontend:** DigitalOcean / Vercel / Netlify.
- **Veritabanı:** Supabase.
